Developed a persistent management tool using OOP &amp; Polymorphism. Features an abstract Worker base class to handle diverse roles via a unified interface. Implements manual heap memory management, file I/O for data persistence, and CRUD logic. Prepared for future $O(1)$ optimization using Hash Maps and STL.

# Worker Management System (C++ OOP)

A robust management tool developed using C++ to demonstrate core Object-Oriented Programming (OOP) principles and data persistence.

## 🛠 Technical Highlights
- **Polymorphism**: Implemented an abstract base class `Worker` to handle diverse roles through a unified interface.
- **Memory Management**: Manual heap allocation and pointer array management.
- **Data Persistence**: Integrated `fstream` for real-time file I/O and local data storage.
- **Clean Architecture**: Separated into `.h` and `.cpp` files for modularity.

## 🚀 Key Features
- Full CRUD operations (Create, Read, Update, Delete).
- Sorting and searching functionality based on unique IDs.
- Input validation and error handling for system stability.
